- 🏗️ At **Conga**, architected the Component Library (Lit.js) and micro-frontend platform — adopted by 40+ production apps, cutting duplicated dev effort by 70%
- 🔐 At **Rocket Mortgage**, built REST APIs for the Identity Platform and an internal MCP server (LangChain + Auth0 + Splunk) that cut auth incident triage from hours to minutes
- 🤖 Built an **AI PR Review Agent** (Spring Boot + LangGraph4j + GPT-4o) — triggers on GitHub webhooks, analyzes diffs for quality and OWASP security, posts structured review comments
- 🧪 Fine-tuned **Qwen 2.5 Coder 7B** via QLoRA → AWQ-quantized → vLLM-served PR reviewer at near-GPT-4o quality for under $8 compute cost

- 📍 San Jose, CA · Open to relocation
